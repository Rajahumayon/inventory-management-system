# Inventory Management System

A simple Inventory Management System built with React.js and Bootstrap.

## Features

- Dashboard Statistics
- Product Management (Add, Edit, Delete)
- Category Management
- Search Products
- Low Stock Alerts
- Inventory Value Calculation
- Export Products to CSV
- Local Storage Support

## Technologies Used

- React.js
- JavaScript
- Bootstrap 5
- Vite
- Local Storage
